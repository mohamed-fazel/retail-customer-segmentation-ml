# 🛒 Retail Customer Segmentation using Machine Learning

## 📌 Project Overview
This project focuses on analyzing an online retail dataset to identify customer segments using machine learning techniques.

The goal is to help businesses understand customer behavior and improve decision-making in marketing and sales.

---

## 📊 Dataset
- File: `Online_Retail.xlsx`
- Contains transactional data of an online retail store

---

## ⚙️ Technologies Used
- Python 
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🚀 Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Scaling
- Customer Segmentation using:
  - K-Means Clustering
  - DBSCAN
- PCA for dimensionality reduction
- Visualization of clusters

---

## 📈 Outcome
- Identified different customer groups based on purchasing behavior
- Useful for targeted marketing strategies

---
